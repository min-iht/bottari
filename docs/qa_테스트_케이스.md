## 0. 실행 전제

- 기본 환경: 데스크톱 Chrome + DevTools 모바일 에뮬레이션 375px. 반응형 케이스(QA-60~)만 3폭 전환.
- "초기 상태"는 localStorage 3개 키가 모두 없는 상태(시크릿 창 또는 초기화 직후).
- 보관함 케이스는 localStorage 모드(config 빈 값)와 Supabase 모드 각각 1회씩 수행한다(표기: [양모드]).
